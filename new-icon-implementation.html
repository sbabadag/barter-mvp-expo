🔄 3. Uygulama Güncellemesi

Icon'u uygulamaya entegre etmek için:

  1. 📁 assets/icon.png dosyasını yeni icon ile değiştirin
  2. 📁 assets/adaptive-icon.png dosyasını da güncelleyin
  3. 🧹 Expo cache'i temizleyin: expo start -c
  4. 🏗️ Yeni build alın: eas build

⚠️ Önemli Notlar:

  • Icon değişikliği sonrası cache temizlemek gerekebilir
  • App Store'larda görünmesi için yeni build gerekir
  • Mevcut icon'un yedeği otomatik alınacak

🚀 4. Son Adımlar

Terminal Komutları:

# Expo cache temizleme
npx expo start --clear

# Development build
eas build --platform ios --profile development
eas build --platform android --profile development

# Production build (App Store için)
eas build --platform ios --profile production
eas build --platform android --profile production
